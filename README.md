# Tasty - Project Documentation

## Project Description
Tasty is a web application that allows users to explore and share their favorite recipes with others. It provides a platform for food enthusiasts to upload, comment, and rate recipes.

## Features
- User authentication (sign up, login, logout)
- Recipe upload and management
- Commenting and rating system
- Search functionality for recipes
- User profiles with saved recipes

## Live Demo
Check out our live demo [here](https://tasty-demo.com).

## Installation Instructions
To install the project locally, follow these steps:
1. Clone the repository: `git clone https://github.com/Nareshaspire/tasty.git`
2. Navigate to the project directory: `cd tasty`
3. Install dependencies: `npm install`
4. Start the development server: `npm start`
